Bold Checkout + commercetools

Minimal frontend integration example

In this example, we showcase a simple way to integrate Bold Hosted Checkout into a commercetools storefront. The approach is simple: create a commercetools access token and cart, then use those (plus a handful of configuration values) to generate a unique checkout URL for Bold Checkout. Finally, redirect the user to the generated URL and Bold Checkout's hosted frontend will take over from there.
